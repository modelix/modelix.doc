Vue.js bindings

Repository | buildfile | Artifacts: Nexus

The @modelix/vue-model-api can be used to create Vue.js applications, that interact with a model server by reading and writing models.

Overview

@modelix/vue-model-api provides Vue.js composable to connect to the model server. It exposes the model as a tree of reactive objects to the application developer. The exposed object can be further wrapped by the generated TypeScript API by the Gradle plugin for typed model API generation.
